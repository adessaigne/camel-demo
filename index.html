What's inside a Camel Exchange ?

  • an input message
    • with body (any Object)
    • with headers (a Map<String, ?>)
  • an output message with body and headers
  • properties (a Map<String, ?>)

How can we manipulate the exchange ?

  • Using producers
  • Using data formats such as CSV
  • Using type conversion such as byte[] to InputSteam
  • Using registered beans
  • Using custom processors

How can we query the exchange ?

For data extraction or conditional execution

Code

Using java DSL

DefaultCamelContext context = new DefaultCamelContext();
context.addRoutes(new RouteBuilder() {
    @Override
    public void configure() throws Exception {
        from("file:inbox?move=.processed")
                .unmarshal().csv()
                .split().body()
                .setHeader("CamelJmsMessageType").constant(JmsMessageType.Map)
                .to("jms:queue:order?connectionFactory=activemq");
    }
});
context.start();

Using Spring DSL

<camelContext id="camel" xmlns="http://camel.apache.org/schema/spring">
    <route>
        <from uri="file:inbox?move=.processed" />
        <unmarshal>
            <csv/>
        </unmarshal>
        <split>
            <simple>${body}</simple>
            <setHeader headerName="CamelJmsMessageType">
                <groovy>org.apache.camel.component.jms.JmsMessageType.Map</groovy>
            </setHeader>
            <to uri="jms:queue:order?connectionFactory=activemq"/>
        </split>
    </route>
</camelContext>
